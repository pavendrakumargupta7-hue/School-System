# School System (Python)

The School System is a Python-based application designed to manage basic school operations efficiently. It helps in handling student records, teacher information, class details, and academic data in an organized manner. The project reduces manual work and improves data management through simple Python logic.

## Features
- Manage student information  
- Store teacher details  
- Handle class and academic records  
- Simple and easy-to-understand structure  

## Technologies Used
- Python

## How to Run
1. Install Python (version 3.x recommended)
2. Clone the repository
3. Run the main Python file using:
   ```bash
   python school_system.py
